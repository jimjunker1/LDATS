LDATS Example

Juniper L. Simonis

This vignette walks through an example of LDATS at the command line and was constructed using LDATS version 0.0.5 on 2018-05-22.

Installation

To obtain the most recent version of LDATS, install the most recent version from GitHub:

install.packages("devtools")
devtools::install_github("weecology/LDATS")

Data

For this vignette, we will be using rodent data from the control plots of the Portal Project, which come with the LDATS package
